# HotelBookGQL
Simple booking API using GraphQL Architectural Style

## Tech Stack
- DotNet Core 6.0
- HotChocolate Framework v12.15.0
- SQL Server 2017 Latest
- Docker

## Tools
- Visual Studio Code
- Docker Desktop
- SSMS

### How to use
1. Clone the project in your local machine
2. Open Docker Desktop 
3. Run docker compose in any CLI
3. Run project
